# th_composer
Small binary that will be used in the [TreasureHunt](https://github.com/treasurehuntgame/treasurehunt) game.
It encodes/decodes strings (as args) or input files, and also has the option to output to a file.

## Techniques
- [x] Morse
- [ ] Reverse
- [ ] RSA cipher
- [ ] Base conversion
- [ ] Keypad code
- [ ] Blowfish cipher
